# 2D 원형 캐릭터 생성 프롬프트 (디테일 강화 버전)

## 기본 설명
원형 프레임 안에 배치된 2D 캐릭터 아바타를 생성해주세요.
캐릭터는 정면을 바라보고 있으며, 얼굴과 상반신이 하나의 단순한 형태로 표현됩니다.

---

## 스타일
- 모던 플랫 디자인 (Modern Flat Design)
- 미니멀 벡터 일러스트 스타일
- 깔끔한 UI 아이콘 스타일
- 부드러운 파스텔 색상 팔레트

---

## 캐릭터 특징
- 피부 톤: Peach tone
- 둥근 얼굴 형태
- 단순화된 얼굴 요소
- 부드럽고 친근한 미소
- 간결한 눈과 눈썹 표현

---

## 헤어 스타일
- 깔끔하고 단순화된 형태의 머리카락
- 과도한 디테일 없이 정돈된 실루엣

---

## 의상
- 단순한 상의 (예: 기본 티셔츠 또는 스웨터)
- 텍스처 없이 평면 색상으로 표현

---

## 색상 팔레트
- 부드러운 파스텔 톤 사용
- Peach 계열 피부색
- 차분한 의상 색상
- 밝고 부드러운 배경색

---

## 구도
- 캐릭터는 원형 프레임 중앙에 배치
- 원형 내부 공간이 균형 있게 구성
- 대칭적이고 깔끔한 레이아웃

---

## 렌더링 스타일
- 플랫 벡터 일러스트 스타일
- 그라디언트는 사용하지 않거나 매우 약하게 사용
- 복잡한 조명 표현 없음
- 그림자는 최소화

---

## 전체 분위기
- 친근하고 따뜻한 느낌의 캐릭터
- 현대적인 UI 아바타 스타일
- 깔끔하고 미니멀한 디자인
